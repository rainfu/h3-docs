# SpellSchool枚举

SpellSchool枚举是VCMI中法术学派的定义，用于区分法术所属的魔法学派。

## 类定义

```cpp
namespace spells
{
    enum class DLL_LINKAGE SpellSchool
    {
        ANY = -1,       // 任意学派
        AIR = 0,        // 气系魔法
        FIRE = 1,       // 火系魔法
        WATER = 2,      // 水系魔法
        EARTH = 3       // 土系魔法
    };
}
```

## 功能说明

SpellSchool是VCMI法术系统中用来表示法术所属魔法学派的枚举。在《英雄无敌》系列中，魔法被分为四个基本元素学派，这个枚举定义了这些学派。法术学派对于法术效果计算、抗性处理和技能修正等方面都很重要。

## 枚举值

- `ANY = -1`: 任意学派，表示不特定于任何学派的法术
- `AIR = 0`: 气系魔法，通常与闪电、风等元素相关
- `FIRE = 1`: 火系魔法，通常与火焰、燃烧等元素相关
- `WATER = 2`: 水系魔法，通常与冰霜、治疗等元素相关
- `EARTH = 3`: 土系魔法，通常与石墙、召唤等元素相关
