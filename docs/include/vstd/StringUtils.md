# StringUtils

## 源文件地址
`e:\develop\heroes\vcmi-assets\vcmi-client\include\vstd/StringUtils.h`

## 概述
`StringUtils`定义了VCMI引擎中的字符串处理工具接口。该文件提供了一系列与字符串操作相关的便捷函数，简化了字符串处理。

## 功能说明

### StringUtils 包含的主要功能：

- 字符串大小写转换的便捷函数
- 字符串分割和连接的辅助方法
- 字符串修剪（去除首尾空白字符）功能
- 字符串格式化和插值的辅助函数
- 字符串比较和查找的便捷函数
- 数字与字符串间的转换功能
