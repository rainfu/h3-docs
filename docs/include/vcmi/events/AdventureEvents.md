# AdventureEvents

## 源文件地址
`e:\develop\heroes\vcmi-assets\vcmi-client\include\vcmi/events/AdventureEvents.h`

## 概述
`AdventureEvents`定义了VCMI引擎中冒险地图相关的事件接口。该文件包含了一系列与冒险地图游戏玩法相关的事件定义。

## 功能说明

### AdventureEvents 包含的主要事件类型：

- 与冒险地图上对象交互的事件
- 英雄移动和探索相关的事件
- 资源收集和管理相关的事件
- 地图探索进度相关的事件
- 冒险地图上战斗触发相关的事件

这些事件用于在冒险地图上发生特定游戏状态变化时通知感兴趣的组件。
