# ApplyDamage

## 源文件地址
`e:\develop\heroes\vcmi-assets\vcmi-client\include\vcmi/events/ApplyDamage.h`

## 概述
`ApplyDamage`定义了VCMI引擎中应用伤害事件的接口。该文件定义了在战斗中应用伤害时触发的事件，允许其他组件拦截或修改伤害计算。

## 功能说明

### ApplyDamage 事件的主要功能：

- 表示在战斗中对单位施加伤害的事件
- 允许其他组件监听并可能修改伤害值
- 提供了伤害计算的可扩展机制，支持各种增益、减益效果
- 使游戏能够实现各种特殊能力或物品效果，这些效果会影响伤害计算

此类事件通常在战斗系统计算出基础伤害后，实际应用伤害前触发，给其他系统一个修改伤害值的机会。
