# Spell

## 源文件地址
`e:\develop\heroes\vcmi-assets\vcmi-client\include\vcmi/spells/Spell.h`

## 概述
`Spell`定义了VCMI引擎中法术接口。该文件定义了游戏中各种法术的通用接口，描述了法术的基本属性和行为。

## 功能说明

### Spell 接口的主要功能：

- 定义法术的基本属性（名称、描述、施法成本等）
- 提供法术效果的查询接口
- 定义法术目标类型和作用范围
- 包含法术学等级要求等相关信息
- 支持法术效果的动态计算和应用

法术接口是VCMI魔法系统的重要组成部分，所有具体的法术实现都遵循此接口定义。
