# Magic

## 源文件地址
`e:\develop\heroes\vcmi-assets\vcmi-client\include\vcmi/spells/Magic.h`

## 概述
`Magic`定义了VCMI引擎中魔法系统相关的接口和枚举。该文件包含魔法类别、法术目标类型等相关定义。

## 功能说明

### Magic 包含的主要功能：

- 定义魔法类别（如破坏性魔法、辅助性魔法等）
- 定义法术目标类型（如对单位施法、对区域施法等）
- 包含魔法效果相关的枚举和常量
- 提供魔法系统的基础类型定义

这些定义为VCMI的魔法系统提供了基础数据结构，用于区分不同类型的法术和魔法效果。
